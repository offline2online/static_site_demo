Templates

Platform-defined display templates for Dynamic Offer Boards. Browse the library and see how each template will look on portrait and landscape displays.
To use a template for a brand, go to Brands → open a brand → Manage Offer Boards → + Add Template.

Stock API Key

Set a secret key that external platforms must include when calling the Stock API. Used as the X-Stock-Api-Key header in API requests.

Store this securely — share only with authorised integrations. See API Reference for usage.

Personalisation Hub API

Configure one or more API environments (e.g. dev, staging, production). The menu board probes each in order and uses the first that responds successfully. Tokens are stored AES-256-GCM encrypted in Firestore.

Controls how the board renders when campaign SKUs are active.

⚠️ Settings use an older single-token format. Please re-enter your API token(s) above and save.
